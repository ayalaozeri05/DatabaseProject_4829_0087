# הוראות הרצה לפרויקט מסד נתונים (שלב 5 - ממשק משתמש)

ברוכים הבאים לפרויקט! קובץ זה מכיל את כל ההוראות הנדרשות על מנת להריץ את המערכת במלואה (כולל מסד הנתונים וממשק המשתמש של שלב 5) בצורה חלקה, פשוטה וברורה.

## 🛠️ דרישות מוקדמות (מה צריך להיות מותקן על המחשב)
לפני שמתחילים, יש לוודא שהתוכנות הבאות מותקנות אצלכם:
1. **Docker Desktop** - חובה שיהיה מותקן ופועל (מופעל ברקע) כדי להרים את מסד הנתונים של PostgreSQL עם כל הנתונים שלנו.
2. **Python** (גרסה 3.8 ומעלה) - נדרש על מנת להריץ את שרת ה-Flask של האפליקציה.

---

## 🚀 שלבי הפעלת המערכת (שלב 5)

אנא עקבו בדיוק אחר השלבים הבאים לפי הסדר:

### שלב 1: הפעלת מסד הנתונים (Docker)
על מנת להרים את מסד הנתונים המכיל את כל הטבלאות, הנתונים, הטריגרים והפרוצדורות שלנו:
1. פתחו חלון **טרמינל (CMD / PowerShell / Terminal)** בתיקייה הראשית של הפרויקט (איפה שנמצא הקובץ `docker-compose.yml`).
2. הריצו את הפקודה הבאה:
   ```bash
   docker-compose up -d --build
   ```
   *(פקודה זו תפעיל את הקונטיינר של מסד הנתונים ברקע. המתינו מעט עד לסיום הפעולה).*

### שלב 2: התקנת ספריות פייתון (Dependencies)
הממשק הגרפי שלנו בנוי בפייתון עם Flask ומשתמש בספריות נוספות כמו `psycopg2` לחיבור למסד הנתונים. כדי להתקין את הכל:
1. באותו טרמינל, היכנסו לתוך תיקיית `phase5`:
   ```bash
   cd phase5
   ```
2. הריצו את הפקודה הבאה להתקנת כל הספריות הדרושות:
   ```bash
   pip install -r requirements.txt
   ```

### שלב 3: הפעלת שרת האפליקציה (Flask)
לאחר שכל הספריות הותקנו ומסד הנתונים רץ:
1. כשהטרמינל עדיין בתוך תיקיית `phase5`, הריצו את קובץ השרת:
   ```bash
   python app.py
   ```
   *(אם הכל עובד כשורה, תראו בטרמינל הודעה שהשרת רץ ומאזין על פורט 5000).*

### שלב 4: כניסה לממשק הגרפי
הכל מוכן! המערכת באוויר:
1. פתחו דפדפן (מומלץ להשתמש ב-Google Chrome או Edge).
2. הכניסו לשורת הכתובות את הלינק הבא:
   [http://localhost:5000](http://localhost:5000)

🎉 **זהו! המערכת עכשיו רצה וזמינה לשימוש ולבדיקה.**

## 💡 דגשים מיוחדים למרצה
- **חיבור לדאטאבייס:** אין צורך לשנות הגדרות חיבור! שרת הפייתון (`db_connection.py`) מוגדר מראש להתחברות חלקה לקונטיינר של הדוקר (localhost בפורט 5433).
- **הרצת שאילתות:** כל השאילתות משלבים קודמים (שלב 2 ושלב 4) הוטמעו בתוך המערכת וניתן להריץ אותן בלחיצת כפתור ישירות ממסך "Queries & Procedures" בממשק.
- **כיבוי וסיום:** בסיום בדיקת הפרויקט, תוכלו לכבות את שרת הפייתון באמצעות לחיצה על `Ctrl+C` בטרמינל. כדי לכבות את מסד הנתונים, חזרו לתיקייה הראשית והריצו בטרמינל:
  ```bash
  docker-compose down
  ```

בדיקה נעימה!
